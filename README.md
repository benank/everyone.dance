# everyone.dance
All code for everyone.dance, a synchronized near-realtime score sharing system for Stepmania.

## Installation
Install the required node modules.
```
npm install
```

## Running

### Run Electron App
```
npm run start
```

### Build Electron App
```
npm run make
```

### Deploy Webapp
```
npm run webapp
```

### Start Server
```
npm run server
```
